# Password Strength Analyzer

This is a **Python-based Password Strength Analyzer** that checks a list of passwords and generates a color-coded HTML report showing their strength.  

It is a beginner-to-intermediate level cybersecurity project that can be used for portfolio purposes.

---

## Features

- Checks password length, uppercase, lowercase, numbers, and special characters.  
- Flags passwords that appear in a list of common passwords.  
- Generates a **color-coded HTML report**:  
  - **Red** → Weak passwords  
  - **Orange** → Medium passwords  
  - **Green** → Strong passwords  

---

## How to Use

1. Ensure you have **Python 3.x** installed.  
2. Clone the repository or download the files.  
3. Open `analyzer.py` in VS Code or any Python editor.  
4. Modify the `password_list` inside the script to test your own passwords.  
5. Run the script:

```bash
python analyzer.py
